Server Requirements

PHP version 5.6 or newer is recommended. It should work on 5.3.7 as well, but we strongly advise you NOT to run such old versions of PHP, because of potential security and performance issues, as well as missing features. MySQL (5.1+) mysqli and pdo drivers

Files & Folder Structure

All files & folders are well organized. You can see below images to get an idea about files & folders. this web app is developed on the top of Codeigniter 3.1.9. So all controllers, models, helpers and other files are located in applicaiton folder. All view files are located inside view folder. Please visit Codeigniter official website for more about files and folders.

Get Started

Now go to inside the extracted folder and browse application -> config folder

to configure database and others. Find config.php, Database.php. files and edit with your editor. Now change following options with your proper value. Please be extra careful to edit the config.php files.

    config.php

  • $config['base_url'] = 'http://yoursite.com/';

    • Please insert your site's right URL or localhost path in base_url, otherwise noting will work.

    database.php

  • 'database' => 'your database'
  • 'username' => 'your database user name',
  • 'password' => 'your database password',

    • Username and password

    The default password for all users is 123456
